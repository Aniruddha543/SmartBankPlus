About Us

We are students from the Institute of Engineering & Management (IEM), Salt Lake, developing SmartBankPlus as part of our DBMS project. It is a secure and functional online banking system designed to demonstrate real-world database management concepts through features like account creation, secure fund transfers, transaction history filtering, PDF statement generation, and user authentication. Built using Supabase as the backend database along with a modern and responsive frontend, SmartBankPlus emphasizes clean database design, data integrity, and efficient query handling while offering a seamless user experience with a professional banking interface.
